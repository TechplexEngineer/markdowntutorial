Autolinking

Github and Reddit will autolink standard URLs so instead of using the common Markdown to format one (and if you don't need a link text or title), you can simply enter it to be turned into a link.

Move through the exercises by using the buttons or the dot menu at the bottom.

Move through the exercises by using the buttons or by swiping left or right.

Play with the concepts you just learned!

For example:
Type an URL with differents formats like: (http://example.com, www.example.com, example.com)
Try other types of URL like mailto:info@example.com or ftp://192.168.0.1 and see if they are turned into links.

See the generated HTML. Even if you don't know HTML, try to understand what's happening. Go wild!
